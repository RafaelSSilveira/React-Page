# React-Page

### Clone
Clone repository

    $ git clone git@github.com:RafaelSSilveira/React-Page.git

## Run with docker-compose
### Build and Run
Install dependecies and run project

    $ docker-compose up --build

Acess http://localhost:3000/


## Run with NPM
### Build
Install dependecies

    $ npm install

### Run
Run project

    $ npm start

Acess http://localhost:3000/
